Hyprland is a 100% independent, dynamic tiling Wayland compositor built from the ground up. It provides the latest Wayland features, powerful plugin support, smooth GPU-accelerated animations, and easy IPC — all while looking stunning out of the box.

## What makes Hyprland different

<CardGroup cols={2}>
  <Card title="Stunning visuals" icon="wand-magic-sparkles">
    Gradient borders, Gaussian blur, drop shadows, and custom bezier-curve animations — all GPU-accelerated.
  </Card>
  <Card title="Fully dynamic workspaces" icon="table-cells">
    Workspaces are created and destroyed on demand. No fixed workspace limit.
  </Card>
  <Card title="Powerful plugin system" icon="plug">
    The built-in `hyprpm` plugin manager lets you install, enable, and update plugins from git repositories.
  </Card>
  <Card title="Hot-reload config" icon="rotate">
    Save your config file and changes take effect instantly — no restart required.
  </Card>
  <Card title="Socket-based IPC" icon="network-wired">
    Control everything programmatically via `hyprctl` or the Unix socket directly.
  </Card>
  <Card title="XWayland support" icon="display">
    Run legacy X11 applications seamlessly alongside native Wayland clients.
  </Card>
</CardGroup>

## Get started in 3 steps

<Steps>
  <Step title="Install Hyprland">
    Follow the [installation guide](/getting-started/installation) for your distribution — packages are available for Arch, NixOS, Gentoo, and many others.
  </Step>
  <Step title="Set up your config">
    Copy the [example config](https://github.com/hyprwm/Hyprland/blob/main/example/hyprland.conf) to `~/.config/hypr/hyprland.conf` and customize it for your setup.
  </Step>
  <Step title="Launch and explore">
    Start Hyprland from a TTY or display manager, then use `hyprctl` to query state and `hyprpm` to install plugins.
  </Step>
</Steps>

title: "IPC events"
description: "Listen to real-time events from Hyprland over socket2 to build scripts, bars, and integrations."
---

Hyprland exposes two Unix domain sockets for IPC:

| Socket | Purpose |
|--------|---------|
| `socket.sock` | Command socket ŌĆö used by `hyprctl` to send queries and dispatchers |
| `socket2.sock` | Event socket ŌĆö streams real-time events as they happen |

Both sockets live at:

```
$XDG_RUNTIME_DIR/hypr/$HYPRLAND_INSTANCE_SIGNATURE/
```

For example:

```
/run/user/1000/hypr/1234abcd5678efgh/.socket.sock
/run/user/1000/hypr/1234abcd5678efgh/.socket2.sock
```

<Tip>
Your `$HYPRLAND_INSTANCE_SIGNATURE` is set automatically when Hyprland starts. Use `hyprctl instances` to list all running instances and their signatures.
</Tip>

---

## Event format

Events emitted on `socket2` use the following line-based format:

```
EVENT>>DATA\n
```

- `EVENT` ŌĆö the event name (lowercase, alphanumeric)
- `>>` ŌĆö separator
- `DATA` ŌĆö event-specific payload (may be empty, a single value, or comma-separated fields)
- `\n` ŌĆö newline terminating the event

Each event is a single line. Newlines in data fields are replaced with spaces.

## Listening to events

<CodeGroup>

```bash socat
socat - UNIX-CONNECT:$XDG_RUNTIME_DIR/hypr/$HYPRLAND_INSTANCE_SIGNATURE/.socket2.sock
```

```bash nc
nc -U $XDG_RUNTIME_DIR/hypr/$HYPRLAND_INSTANCE_SIGNATURE/.socket2.sock
```

```bash shell script
#!/bin/bash
SOCKET="$XDG_RUNTIME_DIR/hypr/$HYPRLAND_INSTANCE_SIGNATURE/.socket2.sock"

socat -U - UNIX-CONNECT:"$SOCKET" | while read -r line; do
    event="${line%%>>*}"
    data="${line#*>>}"
    echo "Event: $event | Data: $data"
done
```

```python python
import socket
import os

sig = os.environ["HYPRLAND_INSTANCE_SIGNATURE"]
runtime = os.environ.get("XDG_RUNTIME_DIR", f"/run/user/{os.getuid()}")
path = f"{runtime}/hypr/{sig}/.socket2.sock"

with socket.socket(socket.AF_UNIX, socket.SOCK_STREAM) as sock:
    sock.connect(path)
    buf = ""
    while True:
        chunk = sock.recv(4096).decode()
        buf += chunk
        while "\n" in buf:
            line, buf = buf.split("\n", 1)
            event, _, data = line.partition(">>")
            print(f"Event: {event!r}  Data: {data!r}")
```

</CodeGroup>

## Emitting custom events

You can emit your own events to `socket2` using the `event` dispatcher. Any connected listener receives them as `custom>>data`.

```bash
hyprctl dispatch event my-event-name
```

This is useful for coordinating between scripts: one process emits an event, another listens on socket2 and reacts.

---

## Example: workspace indicator

A minimal shell script that prints the active workspace whenever it changes:

```bash
#!/bin/bash
SOCKET="$XDG_RUNTIME_DIR/hypr/$HYPRLAND_INSTANCE_SIGNATURE/.socket2.sock"

socat -U - UNIX-CONNECT:"$SOCKET" | while read -r line; do
    event="${line%%>>*}"
    data="${line#*>>}"

    case "$event" in
        workspace)
            echo "Active workspace: $data"
            ;;
        activewindow)
            class="${data%%,*}"
            title="${data#*,}"
            echo "Focused: $class ŌĆö $title"
            ;;
    esac
done
```

## Example: Python event dispatcher

```python
import socket
import os

def connect_to_socket2():
    sig = os.environ["HYPRLAND_INSTANCE_SIGNATURE"]
    runtime = os.environ.get("XDG_RUNTIME_DIR", f"/run/user/{os.getuid()}")
    path = f"{runtime}/hypr/{sig}/.socket2.sock"

    sock = socket.socket(socket.AF_UNIX, socket.SOCK_STREAM)
    sock.connect(path)
    return sock

def iter_events(sock):
    buf = ""
    while True:
        chunk = sock.recv(4096).decode()
        if not chunk:
            break
        buf += chunk
        while "\n" in buf:
            line, buf = buf.split("\n", 1)
            event, _, data = line.partition(">>")
            yield event, data

handlers = {}

def on(event_name):
    def decorator(fn):
        handlers[event_name] = fn
        return fn
    return decorator

@on("workspace")
def handle_workspace(data):
    print(f"Switched to workspace: {data}")

@on("activewindow")
def handle_active_window(data):
    cls, _, title = data.partition(",")
    print(f"Active window: [{cls}] {title}")

with connect_to_socket2() as sock:
    for event, data in iter_events(sock):
        if event in handlers:
            handlers[event](data)
```
